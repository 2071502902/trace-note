# Trace Note PWA 修复版

本包基于 trace_note_mobile_v39_fixed.html 生成，修复 GitHub Pages 中 JavaScript 代码外露的问题。

上传方式：
1. 解压本 zip。
2. 在 GitHub 仓库根目录上传本文件夹内所有内容：index.html、manifest.json、service-worker.js、.nojekyll、icons 文件夹。
3. 覆盖原文件后 Commit changes。
4. 等 1–3 分钟，访问：https://2071502902.github.io/trace-note/?v=repair1

注意：不要上传外层文件夹本身，只上传里面的文件。
